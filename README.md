# Wardle
Wardle is a local 2-player competitive word-guessing game compatible with Windows and Linux. Developed as a first-year university final project, it brings a PvP twist to the familiar Wordle format. 

![Wardle gameplay screenshot](https://github.com/user-attachments/assets/bca82271-cd94-4b1b-bdbc-881ec05b30c9)

To compile the windows version use the .bat, while for the linux version use the terminal commands included in Start.txt.

Thank you for checking it out!

PS: It's also available on itch.io: https://sebastian12p.itch.io/wardle
